
# chapter8-02

Một README cơ bản và ngắn gọn cho dự án nhỏ.

## Mô tả
Dự án minh hoạ nội dung chương 8-02. Mục tiêu: mã dễ đọc, dễ chạy và dễ mở rộng.

## Yêu cầu
- Git
- Node.js (khuyến nghị: >= 14) hoặc ngôn ngữ tương ứng
- npm hoặc yarn

## Cài đặt
1. Clone repository:
```bash
git clone <repo-url>
cd chapter8-02
```
2. Cài đặt phụ thuộc:
```bash
npm install
# hoặc
yarn install
```

## Sử dụng
- Chạy ở chế độ phát triển:
```bash
npm run dev
```
- Build:
```bash
npm run build
```
- Chạy kiểm thử:
```bash
npm test
```

## Cấu trúc thư mục
- src/      — mã nguồn chính
- tests/    — bài kiểm thử
- docs/     — tài liệu bổ sung
- package.json — cấu hình dự án
- README.md — hướng dẫn này

## Đóng góp
1. Fork → tạo nhánh feature → commit → mở Pull Request  
2. Viết test cho chức năng mới nếu cần  
3. Tuân theo style code của dự án
